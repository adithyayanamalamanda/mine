## 🛠️ Built With
* **HTML5** - Semantic and structured layout.
* **Canvas API** - Used for the interactive particle bursts, background gradients, and magnetic cursor effects.
* **CSS3** - Glassmorphism UI, custom animations, variables, and responsive grid layouts.
* **Vanilla JavaScript** - High-performance interactive elements.

## ✨ Features
* **Modern Premium UI:** Dark-mode focused aesthetic with vibrant Cyan, Orange, and Purple neon accents.
* **Interactive Canvas Elements:** Custom mouse-trailer logic, ambient animated background gradient blobs, and click-particle bursts.
* **Dynamic Projects Display:** Features top recent repositories including AgriLink, Webcam Spyware Security, and Mission Placements.
* **Fully Responsive:** Adapts seamlessly to all screen sizes from mobile to ultra-wide displays.
